### 📌 Featured Project

#### 🎓 [PDPU Student Complaint AI](https://github.com/Ict161/student-complaint-ai)

> An end-to-end AI platform that automatically classifies and formalises student complaints using ML + LLMs.

- **ML Model**: LinearSVC + `all-MiniLM-L6-v2` sentence embeddings — **~99% accuracy** on 70K samples
- **AI Rewriting**: Groq LLaMA 3.1 rewrites raw text into formal professional language
- **Full-Stack**: React 18 + Vite frontend, FastAPI + SQLite backend
- **12 department routing**, 4-level urgency detection, role-based dashboards
- **Stack**: Python · FastAPI · React · Tailwind · scikit-learn · Groq API
